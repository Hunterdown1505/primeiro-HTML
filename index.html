O que é o inspetor de elementos ?

(Voltar)

Basicamente, o inspetor de elementos é uma ferramenta para o developer, ou seja, algo que se utiliza para ver especificação de páginas web o que, posteriormente, ajuda a quem está querendo ingressar no mercado de trabalho na Web Developer, pois pode ser considerado uma sala de aula no navegador. Vale ressaltar que, o inspetor de elementos é uma ferramenta oferecida pelo navegador e não do HTML.

Estruturas básicas do HTML.

(Voltar)

As estruturas básicas do HTML são o início de tudo, pois são a base de todos os HTML que são criados e, futuramente, irão tornar algo mais bem trabalhado e maior. As estruturas básicas do HTML são as seguintes :

  • HTML: Define o início do documento HTML.
  • HEAD: Define informações sobre o documento, tal como título e links.
  • BODY: Tudo que o usuário terá acesso visual, ou seja, tudo que ele poderá ver ou interagir.

Lembrando que o HTML possuí inúmeras outras funções, mas essas são totalmente ESSENCIAIS para o funcionamento dele.

O que são as TAGs ?

(Voltar)

As TAGs são elementos usados para estruturar e formatar o conteúdo de uma página web. Elas são compostas por um nome entre os sinais de menor (<) e maior (>) e são usadas em pares, uma de abertura e outra de fechamento, com uma barra (/) antes do nome da tag de fechamento.

Estas mesmas TAGs contêm atributos com formas mais específicas, mas veremos isso tópico a seguir.

O que são atributos ?

(Voltar)

Os atributos são como uma extensão das TAGs ou, se preferir, pode tratá-los como especificações das mesmas. São responsáveis por formatar o texto ou interagir o usuário de forma mais específica tornando assim, as páginas da mais dinâmicas e funcionais. Alguns exemplos de atributos estão listados abaixo :

  • Id = Atributo utilizado para criar uma identificação em algum parágrafo, tópico ou título/subtitulo específico.
  • href = Atributo utilizado na tag de ancoragem e utilizado para especificar o URL para qual o link deve apontar.

Textos

(Voltar)

Os textos referem-se ao conteúdo textual que é exibido na página web. Eles podem incluir títulos, parágrafos, listas, links, entre outros elementos que contenham texto visível para os usuários. O texto é uma parte essencial do conteúdo de uma página web, fornecendo informações, instruções e interatividade para os usuários.

Listas ordenadas e não ordernadas.

(Voltar)

Listas ordenadas (ol) são listas em que os itens são numerados sequencialmente ou com outro tipo de marcador específico, como letras ou romanos. Já as listas desordenadas (ul) são listas em que os itens são marcados com símbolos, como pontos, quadrados ou círculos. Ambos os tipos de listas são úteis para organizar e apresentar informações de forma estruturada em uma página web. Ambas as listas podem ser modificadas no CSS com símbolos diferentes ou coisas do tipo (Talvez uma logo de marca ou algo do tipo). Links

(Voltar)

No HTML, os hyperlinks são criados usando a tag "a" (âncora). Eles servem para guiar os usuários para outras páginas da web, recursos ou seções da mesma página. O atributo href é inserido dentro da tag "a" para indicar o destino do hyperlink, que pode ser um URL ou um marcador dentro do mesmo documento. Quando alguém clica em um hyperlink, o navegador segue o URL especificado ou desloca-se até a parte da página indicada pelo marcador.
